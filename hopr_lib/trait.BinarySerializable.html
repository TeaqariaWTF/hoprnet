Trait hopr_lib::BinarySerializable

source · 
pub trait BinarySerializable: Sized {
    const SIZE: usize;

    // Required methods
    fn from_bytes(data: &[u8]) -> Result<Self, GeneralError>;
    fn to_bytes(&self) -> Box<[u8]>;
}
Expand description

A type that can be serialized and deserialized to a binary form. Implementing this trait automatically implements ToHex trait which then uses the serialize method.

Required Associated Constants§

source

const SIZE: usize

Minimum size of this type in bytes.

Required Methods§

source

fn from_bytes(data: &[u8]) -> Result<Self, GeneralError>

Deserializes the type from a binary blob.

source

fn to_bytes(&self) -> Box<[u8]>

Serializes the type into a fixed size binary blob.

This trait is not object safe.
